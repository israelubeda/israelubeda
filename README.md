## Sobre mí

Desarrollador chileno con base en **Santiago, Chile**, activo en GitHub desde 2018. Me especializo en construir herramientas prácticas con **Python y Flask**, con foco en el ecosistema tributario chileno (SII, DTE, RUT). En los últimos años he expandido mi interés hacia la **inteligencia artificial, LLMs y agentes autónomos**.

- 🏗️ Construyo herramientas tributarias y fiscales para el mercado chileno
- 🤖 Explorando IA generativa, agentes autónomos y el ecosistema Claude
- 🌐 Comparto tutoriales técnicos en mi [blog personal](https://israelubeda.github.io/)
- 📍 Santiago, Chile

## Proyectos Destacados

### 🧾 [xml_to_pdf_DTE_Chile](https://github.com/israelubeda/xml_to_pdf_DTE_Chile)
Convierte facturas electrónicas XML del **SII chileno** a PDF con formato profesional. Incluye códigos de barra, plantillas personalizables y descarga automática.
`Python` · `Flask` · `WeasyPrint` · `Bootstrap 5` · `Jinja2`

---

### 📅 [inhabiles](https://github.com/israelubeda/inhabiles)
Calcula automáticamente los **días inhábiles en Chile** (fines de semana + feriados oficiales). Útil para planificación de vacaciones y gestión de RRHH.
`Python` · `Flask` · `holidays`

---

### 🪪 [modulo_11](https://github.com/israelubeda/modulo_11)
Verificador de **RUT chileno** con algoritmo módulo 11. Interfaz web responsiva para validar RUTs en tiempo real.
`Python` · `Flask` · `Bootstrap 5`

---

### 🖼️ [meme-generator](https://github.com/israelubeda/meme-generator)
Generador de memes web: sube una imagen, añade texto superior e inferior y descarga el resultado.
`Python` · `Flask` · `Pillow`

---

### 🎨 [color_es](https://github.com/israelubeda/color_es)
Librería Python para cambiar el color del texto en consola, con la API en **español**.
`Python`
